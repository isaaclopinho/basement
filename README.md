## Iniciando

Primeiro, instalar as dependências do projeto:

```bash
    yarn
```

Adicione as variáveis de ambiente como citado em .env.example

E por fim, execute o projeto:

```bash
    yarn dev
    or
    yarn build
    yarn start
```
